레일픽(RailPick) 개인정보처리방침

적용일: 2025-11-29

본 개인정보처리방침은 레일픽(RailPick) 애플리케이션(이하 "앱")이 제공하는 서비스와 관련하여, 사용자의 개인정보가 어떻게 수집·이용·보관·보호되는지를 설명합니다. 본 앱을 이용함으로써 본 개인정보처리방침에 동의하는 것으로 간주됩니다.

1. 수집하는 개인정보 항목

앱은 다음 정보를 수집할 수 있습니다.

  • 기기 정보: 기기 모델, 운영체제 버전, 광고 식별자(Advertising ID)
  • 앱 사용 정보: 화면 이동 기록, 기능 사용 로그, 조회 패턴 정보
  • 오류 및 진단 정보: Firebase Crashlytics를 통한 크래시 로그 및 오류 분석 정보
  • 원격 설정 정보: Firebase Remote Config를 통해 원격으로 제공되는 설정 값
  • 광고 관련 정보: Google AdMob이 수집하는 광고 노출/클릭 관련 데이터
  • AI 최적화 관련 기술 데이터(익명 처리됨): 조회 간격, 세션 지속 시간, 성공/실패 여부 등 — Cloud Run·BigQuery 기반 AI 모델 고도화를 위해 사용

본 앱은 회원가입 기능을 제공하지 않으며, 사용자가 직접 입력하지 않는 개인식별 정보(이름, 전화번호, 결제정보 등)는 수집하지 않습니다.

2. 개인정보 수집 목적

앱은 다음 목적을 위해 개인정보를 수집·이용합니다.

  • 앱 핵심 기능 제공
  • 앱 안정성 확보 및 오류 분석
  • 광고 제공 및 광고 성능 분석(AdMob)
  • 사용자 환경 개선을 위한 통계 분석 및 UI/UX 향상
  • Cloud Run + BigQuery 기반 AI 조회 품질 향상

AI 분석에 사용되는 데이터는 모든 개인 식별 요소가 제거된 비식별 기술 데이터이며, 역·시간·조회 패턴과 같은 기술적 요소만 포함됩니다.

3. 제3자 서비스 제공자

앱은 다음 제3자 서비스를 사용합니다.

  • Firebase Analytics – 앱 사용 통계 분석
  • Firebase Crashlytics – 앱 오류 및 크래시 분석
  • Firebase Remote Config – 기능 설정값 원격 업데이트
  • Google AdMob – 광고 게재
  • Google Cloud Run + BigQuery – AI 최적화 처리를 위한 익명화 기술 데이터 분석

광고 제공을 위해 AdMob은 광고 식별자(Advertising ID) 등 정보를 수집할 수 있습니다. 앱 ID는 AndroidManifest에 포함되어 있으며, 광고 단위 ID(ad unit ID)는 원격 설정(Remote Config)으로 동적으로 제공될 수 있습니다.

각 제3자 서비스의 개인정보처리방침은 아래에서 확인할 수 있습니다.

  • Google Privacy Policy

4. 개인정보의 보관 기간

앱은 서버에 사용자의 개인정보를 직접 저장하지 않습니다.
Firebase, AdMob, Cloud Run, BigQuery 등 제3자 서비스가 수집하는 데이터는 각 서비스의 정책에 따라 보관됩니다.
