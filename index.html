레일픽(RailPick) 개인정보처리방침

적용일: 2025-11-29

본 개인정보처리방침은 레일픽(RailPick) 애플리케이션(이하 “앱”)이 제공하는 서비스와 관련하여, 사용자의 정보가 어떻게 수집·이용·보관·보호되는지를 설명합니다. 본 앱을 이용함으로써 본 개인정보처리방침에 동의하는 것으로 간주됩니다.

1. 수집하는 정보 항목

앱은 다음의 비식별 기술 정보를 수집할 수 있습니다.

  • 기기 정보: 기기 모델, 운영체제 버전, 광고 식별자(Advertising ID)
  • 앱 사용 정보: 화면 이동 기록, 조회 시도 정보, 기능 사용 로그
  • 오류 및 진단 정보: Firebase Crashlytics의 크래시 로그 및 오류 분석 데이터
  • 원격 설정 정보: Firebase Remote Config를 통해 전달되는 기능 설정값
  • 광고 관련 정보: Google AdMob이 수집하는 광고 노출·클릭 관련 데이터
  • AI 최적화를 위한 익명 기술 데이터: 조회 간격, 세션 지속시간, 성공/실패 여부 등 — Cloud Run 및 BigQuery 기반 모델 개선에 활용

앱은 회원가입 기능을 제공하지 않으며, 사용자를 식별할 수 있는 개인정보(이름, 전화번호, 결제정보 등)는 수집하지 않습니다.

2. 정보 수집 목적

앱은 다음 목적을 위해 정보를 이용합니다.

  • 열차 조회 및 예약 기능 제공
  • 앱 안정성 확보 및 오류 분석(Firebase Crashlytics)
  • 광고 제공 및 광고 성능 분석(Google AdMob)
  • 사용자 이용 패턴 분석 및 UI/UX 개선
  • Cloud Run + BigQuery 기반 AI 최적화 기능 개선

AI 분석에 사용되는 모든 데이터는 비식별 처리되며, 개인을 특정할 수 없습니다.

3. 제3자 서비스 제공자

앱은 다음 제3자 서비스를 사용합니다.

  • Firebase Analytics – 앱 사용 통계 분석
  • Firebase Crashlytics – 오류 및 크래시 분석
  • Firebase Remote Config – 기능 설정 원격 업데이트
  • Google AdMob – 광고 제공 및 광고 개인화 처리
  • Google Cloud Run / BigQuery – 익명 기술 데이터 기반 AI 모델 분석

광고 제공을 위해 AdMob은 광고 식별자(Advertising ID)를 사용하며, 광고 노출·클릭 정보 등을 수집할 수 있습니다.

각 제3자 서비스의 개인정보처리방침은 다음에서 확인할 수 있습니다.

  • Google Privacy Policy

4. 정보 보관 기간

앱은 사용자의 개인정보를 자체 서버에 저장하지 않습니다.
Firebase, AdMob, Cloud Run, BigQuery 등 제3자 서비스가 수집하는 데이터는 각 서비스의 정책에 따라 보관됩니다.

5. 이용자의 선택 및 권리

앱은 개인을 식별할 수 있는 개인정보를 수집하지 않기 때문에 법률상 “개인정보 열람·정정·삭제” 권리는 적용되지 않습니다. 다만 사용자는 다음과 같은 선택을 할 수 있습니다.

  • 데이터 전송(업로드) 동의 철회: 앱 설정에서 “데이터 전송(업로드)” 기능을 비활성화할 수 있습니다.
  • 광고 개인화 거부: 기기 설정 → Google → 광고에서 개인화 광고 수신을 해제할 수 있습니다.

안내: Cloud Run·BigQuery에 저장되는 데이터는 모두 비식별 상태이므로 개별 사용자를 특정하여 삭제하는 것은 불가능합니다.
